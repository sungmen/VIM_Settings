# VIM_Settings

```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
<br>
set vimrc

```
vi ~/.vimrc
```
<br>
VIM_SETTINGS repository .vimrc download command

```
wget https://raw.githubusercontent.com/sungmen/VIM_Settings/master/.vimrc
```
